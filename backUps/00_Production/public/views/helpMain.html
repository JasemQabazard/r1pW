Start Here First

  • The simplest way to start using read1page.com on your first visit is to press the READ button on the home page
  • You will join all site users in a common Holly Quran recital; code name: "KHATMA".
  • when you press READ you receive al-fatiha and one more page of the holly Quran to read
  • once you read this one page you would have participated in a shared recital with everyone else
  • your contribution was this one page.
  • As you visit read1page.com later and press "READ"", you can continue to contribute to this common recital
  • UNLESS YOU CHANGE THE RECITAL CODE, to other copmmon recitals or to your own recital code

If you change the recital code

  • Once you press READ you will participate in the recital code you changed to.
  • See the enter recital code menu option 1, on this Help to find out how to change the recital code

The progress bars at the footer and the read Khatma buttons

  • The progress bars at the buttom indicate the progress in completing 3 KHATMAS
  • namely:
  • "KHATMA" A recital to read al-fatiha and ONE more page
  • "KHATM2" A recital to read al-fatiha and TWO more pages

Option Menu Items

Option Menu One - Change Recital Code

  • This is a very simple form to be used to change your current recital code
  • The form has two fields to enter the same recital code twice for added security
  • Then press SEND
  • Once you change your recital code, the READ button on the home page will provide you pages according to the sequential readings of your current recital code
  • You can change back to the common recital code "KHATMA", "KHATM2", "KHATM5", or any other recital code
  • You may also receive a recital code from a friend to join his recital or you may decide to generate your own recital code from menu option 2

Option Menu Two - Generate New Recital Code

  • You can use this form to generate a new recital code, then share it with your friends or family
  • Although we email your recital code to you when you generate it, Please remember your code and keep your email on record since we do not keep recital codes attached to your name in any way and we do not store your Email ID
  • To generate a new recital code:
  • Enter the number of pages you want for every press of the READ button on the home page, you can ask for 1, 2, or 5 sequential pages to be generated on every READ press
  • Select if you would like or not, Al Fatiha as the first verse on every press of the READ button
  • Supply your email twice, so that you receive an email with the newly generated recital code for safe keeping
  • Press Generate
  • Once you generate a new recital, your current recital code on the web site will be changed to the newly generated code
  • The email will be sent to you, remember we do not keep your Email ID on store

  • What is the total number of generated recital codes
  • How many recitals were completed up to the moment

  • How many Al Fatihas were read to date

Read Navigation Buttons


  • These navigation buttons will help you complete your reading of the pages supplied to you by the site when you press READ
  • The left button advances you to the next page to be read, if any
  • The right button takes you back to the previous page, if any
  • Once you complete reading all the pages supplied to you, you must press the check mark (green) to tell the web site that these pages have been completely read
  • If you do not complete reading your pages or read some of them, you can exit the web site and return later to complete your reading. ie the current pages will be kept for you unless of course you press the check mark (green) button
